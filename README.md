**프로젝트 개요**
 - 팀원 소개
   - 김동재 : 호스트 - 클라이언트 구조에서의 동기화 기능 구현, 게임 로직 구현
   - 엄장헌 : 게임 로직 및 클라이언트 메인 개발
   - 양정우 : 게임 로직 및 클라이언트 서브 개발
   - 임윤수 : 게임 기획 및 리소스 제작
 - 개발 기간: 2025.04 - 2026.02 
 - 기술 스택: Unity, Fusion 2.0.5  

***리포지토리 안내***

본 프로젝트는 **Unity Version Control(UVC)을 기반으로 관리**되었으며, 본 GitHub 리포지토리는 개발 당시 
기획 문서 및 리소스 공유를 위한 목적으로 운용되었습니다. 
따라서 전체 프로젝트의 소스 코드는 포함되어 있지 않습니다. 
다만 엔지니어링 역량 증명을 위해 프로젝트의 핵심 로직을 담은 주요 스크립트들을 별도의 폴더로 발췌하여 공개하고 있습니다.

**기술적 도전**
 - Photon Fusion2 : Unity 기반, 협동 게임이라는 환경에 맞춰 기존에 사용 경험이 없던 Photon Fusion2를 학습 및 활용하여
                    멀티플레이 기능을 구현하였습니다.

**문제 해결 사례**
 - 문제 상황 : 플레이어가 점프키를 연타하면 캐릭터의 상태가 Jump에 고정되어 전이되지 않는 문제가 발생했습니다.
 - 해결 방안
   - 상태 전이 구조 재설계 : 기존의 이벤트 의존적 상태 전이 방식에서 벗어나, 논리적 흐름이 보장되는 명시적 전이 구조를 구축했습니다.
   - 계층별 권한 분리 : 로직 계층과 표현 계층의 권한을 분리하여 네트워크 환경에서도 상태 전이의 일관성을 확보했습니다. 

**주요 코드 안내**
CoreScripts 폴더 내에 프로젝트의 핵심이 되는 네트워크 동기화 및 상태 제어 관련 코드들이 업로드되어 있습니다. 
각 코드 내 주석을 통해 상세한 설계 의도를 확인하실 수 있습니다.
